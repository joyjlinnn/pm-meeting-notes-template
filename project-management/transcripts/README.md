# Transcripts

Drop raw meeting transcripts here (optional). The agent can read from this folder when you ask it
to process a meeting.

**Naming convention:** `<Topic> - Transcript YYYY-MM-DD.md`

Example prompt in Cursor:

> "Process the transcript in `project-management/transcripts/` and update the meeting summary,
> open questions, and next steps."

You can also paste a transcript directly into chat without saving a file first.
