Living tracking docs for a **sub-workstream** under `project-management/`. Use subfolders when a
program has a distinct track (e.g. message bus, billing, onboarding).

## Cursor Canvas

To use the interactive canvas in Cursor:

1. Copy `message-bus-missing-events-status.canvas.tsx` to  
   `~/.cursor/projects/<your-workspace>/canvases/`
2. Open the file in Cursor → **Open Canvas**

Or open the **Preview.html** in any browser for a static visual with copy buttons.
